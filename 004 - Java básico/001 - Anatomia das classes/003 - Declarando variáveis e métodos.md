## Declarando variáveis e métodos

Como declarar cada um:

### Variável

```java
// estrutura

tipo nome = valor (pode ser opcional);

// exemplo
int idade = 23;
```

### Método

```java
// estrutura

tipoDoRetorno NomeObjetivoNoInfinitivo ( Parametro(s) )

// exemplo

int somar (int numero1, int numero2)

String formatarCep ( long cep )

```
